Objetivos del Proyecto

Objetivo General

Desarrollar una estrategia integral para la validación, modelado predictivo y visualización de concentraciones de PM2.5 y PM10 mediante sensores de bajo costo, como complemento a las redes oficiales de monitoreo de calidad del aire en Bogotá.

Objetivos Específicos

1. Validación de Sensores

Validar el desempeño de sensores de bajo costo (PMS5003) mediante comparación con datos de referencia de la Red de Monitoreo de Calidad del Aire de Bogotá (RMCAB).

2. Modelos de Calibración

Desarrollar modelos de calibración utilizando algoritmos de Machine Learning (Regresión Lineal, Random Forest, Support Vector Regression) que mejoren la precisión de las mediciones considerando variables ambientales como temperatura y humedad relativa.

3. Plataforma Web Interactiva

Implementar una plataforma web interactiva para la visualización de datos históricos, comparación de modelos y análisis de tendencias de contaminación atmosférica en Bogotá.

4. Evaluación de Viabilidad

Evaluar la viabilidad técnica y económica del uso de sensores de bajo costo como complemento a las estaciones oficiales de monitoreo en contextos urbanos.
