¿Qué es este proyecto?

Una solución innovadora para ampliar el monitoreo de calidad del aire en Bogotá

Validación

Calibración rigurosa de sensores de bajo costo (PMS5003) contra datos de referencia de la RMCAB (Red de Monitoreo de Calidad del Aire de Bogotá)

Modelado Predictivo

Aplicación de algoritmos de Machine Learning: Regresión Lineal, Random Forest, Support Vector Regression para mejorar la precisión de las mediciones

Visualización

Plataforma web interactiva para explorar datos históricos, comparar modelos y analizar tendencias de contaminación atmosférica

El Problema

La contaminación del aire es uno de los mayores riesgos ambientales para la salud pública en ciudades como Bogotá.

  • Cobertura limitada: Las redes oficiales de monitoreo son costosas y tienen baja densidad espacial
  • Exposición desigual: Muchas zonas urbanas carecen de información sobre calidad del aire
  • Impacto en salud: PM2.5 y PM10 causan enfermedades respiratorias y cardiovasculares
Nuestra Solución

Utilizamos sensores de bajo costo como complemento a las estaciones oficiales, combinando:

  • Tecnología accesible (sensor PMS5003)
  • Calibración con métodos de Machine Learning
  • Transmisión remota de datos en tiempo real
  • Visualización web interactiva y accesible

Basado en evidencia científica de estudios en Barcelona, Medellín, Bogotá y Pamplona
